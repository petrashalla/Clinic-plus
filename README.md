# Верстка главной страницы клиники


### Стек

- **HTML**
- **SCSS**
- **JS**
- **Gulp** 


### Команды для сборки и развертывания

Установить папку node_modules:
```
npm run instal
```

Установить зависимости:
```
npm i
```

Зпустить в режиме разработки:
```
gulp
```
